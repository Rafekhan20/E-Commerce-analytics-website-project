Connecting Power BI to your E-Commerce Data

Bring the sqlite database data directly into Power BI Desktop for advanced modeling, interactive filtering, and scheduled reports. Follow these steps to build your workspace.

1

Export Database to CSV

We've created a Python script that exports the SQLite tables into optimized, clean CSV files for easy loading in Power BI.

python power_bi/export_csv.py
2

Load Data & Setup Relationships

Import the CSVs into Power BI Desktop and create a Star Schema layout:

  • Orders[user_id] → Users[user_id]
  • OrderItems[order_id] → Orders[order_id]
  • OrderItems[product_id] → Products[product_id]
  • WebEvents[user_id] → Users[user_id]
3

Create DAX Measures

Use these DAX queries to compute advanced analytics inside Power BI:

Average Order Value (AOV): AOV = DIVIDE(SUM(Orders[total_amount]), DISTINCTCOUNT(Orders[order_id]))
Year-to-Date Revenue: YTD Revenue = TOTALYTD(SUM(OrderItems[quantity] * OrderItems[unit_price]), 'Calendar'[Date])

Generate Power BI Dataset Files

Run the exporter now to write the data tables to CSV inside your workspace directory.
